Rexster is a graph-based recommender engine. This engine makes use of graph traversal to rank and score elements in a graph. A REST-ful interface is provided into to allow Rexster to work as a standalone server.

The documentation for Rexster can be found at this "location":http://rexster.tinkerpop.com. Finally, please visit "TinkerPop":http://tinkerpop.com for other software products.
